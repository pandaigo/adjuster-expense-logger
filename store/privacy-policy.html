Privacy Policy

Last updated: 2026-05-11

Adjuster Expense Logger ("the Extension") is a Chrome browser extension that helps independent insurance adjusters log per-claim deployment expenses (per diem, hotel, mileage, meals, parking, supplies, phone, other) and export them as CSV or PDF for IA company billing and personal tax records.

Data Collection

The Extension does not collect, transmit, or share any personal data. Specifically:

  • No data is sent to any external server.
  • No analytics, tracking, or telemetry is included.
  • No user accounts are required.
  • No cookies are set by the Extension.

Data Storage

All expense entries, deployment information (adjuster name, CAT event, start / end dates), Pro purchase status, and the IRS mileage rate preference are stored locally on the user's device using the Chrome Storage API (chrome.storage.local). This data never leaves the device unless the user manually exports it as a CSV / PDF / JSON file.

Permissions

The Extension requests the following permissions:

  • storage — To save expense entries, deployment details, and preferences locally on the device.
  • host access to https://extensionpay.com/* (declared via content_scripts in the manifest) — Required by ExtensionPay (our payment processor) to detect when a one-time Pro purchase completes. The ExtPay content script runs only on the ExtensionPay checkout page. Only purchase status (paid / unpaid) is exchanged. No expense data, claim numbers, or deployment details are shared.

Third-Party Services

The Extension uses ExtensionPay (extensionpay.com) to process one-time payments and verify purchase status. When a user upgrades to the Pro version, payment is handled by ExtensionPay via Stripe. Only purchase status (paid / unpaid) is shared with the Extension. No expense entries, claim numbers, deployment data, or other adjuster information is shared with ExtensionPay, Stripe, or any other third party.

Limited Use Disclosure

Use of information received from Chrome Web Store APIs and any Google user data will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Disclaimer

The Extension is a personal record-keeping tool. It does not provide tax, accounting, or legal advice. Users should verify exported data with their CPA, IA company, and the current IRS rules before relying on it for tax filings, billing disputes, or any regulatory submission. No warranty, express or implied, is provided.
